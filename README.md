# 🌟 **AceGates – AG | מאגר ערכות קלפי אסים מאירים**

מערכת מהודרת שתאסוף _עשרות_ עיצובים של **קלפי אס (Aces)** —  
כל אחד מהם משמש **שער של אור**, “אס = אין־סוף הארות”,  
והם מתוכננים לשימוש במשחקים, לימוד, מדיטציה, יצירה רוחנית, ולכל פרויקט SparKing.

## 🔥 **מבנה המאגר**

כל ערכה = **4 אסים** (רוח, אש, מים, אדמה)

- **אס חמישי נסתר** (אור/אמת/עולם/לב/נשמה)  
    כל אס = שער אנרגטי עם מסר, צבע, סמל וחותם פנימי.  
    כל עיצוב מגיע בגרסאות:  
    ✨ מיסטי • ✨ הולוגרפי • ✨ כרום־זהב • ✨ ספירלה קוסמית • ✨ אש-רוח-מים-אדמה
